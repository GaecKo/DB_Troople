Pour un mini projet, je dois faire une interface en html / js / css qui permet de créer une DB. Le but serait d'avoir une page principale, avec un bouton pour ajouter une nouvelle colonne. Un pop up s'afficherait alors permettant de créer cette colonne, où on peut choisir le type, une description, ...
Sur la page principal, nous aurions un visu sur les différentes colonnes créés, on devrait pouvoir les modifier, et finalement il devrait y avoir un bouton exporter qui permettrait de récupérer une version json de nos colonnes une par une avec les caractéristiques etc. J'aimerais que tu fasses le code, n'hésite pas à créer des classes en JS, et à rajouter pas mal de style CSS. J'aimerais aussi à l'avenir pouvoir importer un fichier json correspondant à une db (comme la sortie que nous faisons quand on exporte) et avoir son visuel en colonne sur la page principale
